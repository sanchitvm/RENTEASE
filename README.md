# RentEase

RentEase is a modern MERN stack real estate application designed to simplify property rentals and management.

## Project Structure

- **Client**: React + Vite frontend with Tailwind CSS.
- **Server**: Node.js + Express backend with MongoDB.

## Key Features

- **Verified Listings**: High-quality, verified room and property listings.
- **Secure Agreements**: Digital rental agreements for seamless transactions.
- **Integrated Payments**: Secure payment processing with Razorpay.
- **Real-time Chat**: Direct communication between landlords and tenants.

## Getting Started

### Prerequisites

- Node.js
- MongoDB

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sanchitvm/RENTEASE.git
   ```

2. Install dependencies:
   - Client: `cd Client && npm install`
   - Server: `cd Server && npm install`

3. Set up environment variables in `Server/.env`.

4. Run the application:
   - Client: `npm run dev`
   - Server: `npm start` (or `nodemon index.js`)
